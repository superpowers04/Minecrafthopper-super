Minecraft Crash Help

Exit Code: -805306369

This exit code is caused when the client runs out of memory.

This can be caused by the JVM not being allocated enough memory, a bug in the game, using client modifications or a dependency of the game that has unexpectedly terminated.

How to fix this

  • See this article on instructions to allocate more memory.
  • Reinstall minecraft to ensure you have no client mods.
  • Check your System Event Log for issues.
